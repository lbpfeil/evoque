<task type="auto">
  <name>Task 1: Adjust light mode to warmer cream/beige tones</name>
  <files>index.css</files>
  <action>
**Current light mode values (too white):**
```css
--background: oklch(0.995 0.003 90);  /* L=99.5% - nearly pure white */
--card: oklch(1 0.002 85);             /* L=100% - pure white */
--sidebar: oklch(0.98 0.005 85);       /* L=98% - very bright */
```

**New values (warmer cream/beige):**

Update :root in index.css with these adjusted values:
```css
--background: oklch(0.97 0.015 85);    /* Warmer cream, slight yellow-orange hue */
--foreground: oklch(0.18 0.01 50);     /* Keep as-is */
--card: oklch(0.98 0.012 80);          /* Slightly warmer white for cards */
--card-foreground: oklch(0.18 0.01 50); /* Keep as-is */
--popover: oklch(0.98 0.012 80);       /* Match card */
--popover-foreground: oklch(0.18 0.01 50); /* Keep as-is */
--secondary: oklch(0.93 0.015 75);     /* Warmer secondary */
--muted: oklch(0.94 0.012 78);         /* Warmer muted */
--accent: oklch(0.92 0.05 70);         /* Warmer accent */
--border: oklch(0.88 0.015 70);        /* Slightly more visible border */
--input: oklch(0.88 0.015 70);         /* Match border */

/* Sidebar - match background for consistency */
--sidebar: oklch(0.97 0.015 85);       /* Same as --background */
--sidebar-border: oklch(0.88 0.015 70); /* Same as --border */
```

**Color explanation:**
- Lowered L (lightness) from 99.5% to 97% for visible warmth
- Increased C (chroma) from 0.003 to 0.015 for more color saturation
- Hue 80-85 gives a warm cream/beige tone (yellow-orange family)
- Sidebar now matches background for visual consistency
  </action>
  <verify>
1. Run `npm run dev` and view app in light mode
2. Background should have visible cream/beige tint, not stark white
3. Compare against pure white (#fff) - should be noticeably warmer
4. Sidebar and main content should have same background color
  </verify>
  <done>Light mode has warm cream/beige tones instead of stark white</done>
</task>
